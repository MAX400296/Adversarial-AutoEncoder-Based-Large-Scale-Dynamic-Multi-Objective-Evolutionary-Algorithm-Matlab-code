## 📘 项目简介

本仓库包含论文的官方实现：

> **Adversarial AutoEncoder-Based Large-Scale Dynamic Multi-Objective Evolutionary Algorithm**  
> *李晨阳, Gary G. Yen, 何振安*  
> 发表在 *IEEE Transactions on Evolutionary Computation, 2024*  
> DOI: [10.1109/TEVC.2024.3412049](https://doi.org/10.1109/TEVC.2024.3412049)

本项目提出了一种基于 **对抗自编码器（AAE）** 的深度生成框架，用于求解**大规模动态多目标优化问题（DMOPs）**。  
该方法结合生成模型与进化优化算法，在环境动态变化时能**自适应地产生高质量初始种群**，尤其适用于**高维搜索空间（最高至1000维）**。
