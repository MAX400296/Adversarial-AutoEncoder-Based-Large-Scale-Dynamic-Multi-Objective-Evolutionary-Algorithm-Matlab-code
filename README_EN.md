## Overview

This repository contains the official implementation of the paper:

> **Adversarial AutoEncoder-Based Large-Scale Dynamic Multi-Objective Evolutionary Algorithm**  
> *Chenyang Li, Gary G. Yen, and Zhenan He*  
> Accepted in *IEEE Transactions on Evolutionary Computation, 2024*  
> DOI: [10.1109/TEVC.2024.3412049](https://doi.org/10.1109/TEVC.2024.3412049)

This project introduces an **Adversarial AutoEncoder-based (AAE)** deep generative framework for solving **Large-Scale Dynamic Multi-Objective Optimization Problems (DMOPs)**.  
The framework integrates a generative model with evolutionary optimization to **adaptively generate high-quality initial populations** under dynamic environmental changes â€” particularly effective in **high-dimensional (up to 1000D)** search spaces.
